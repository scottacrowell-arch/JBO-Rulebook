# Westside Rules Hub v1.6

Mobile-first GitHub Pages prototype.

## v1.6 changes

- Removed the floating Popular Rules section from the Home page.
- Added a dedicated Popular Rules page with a clean table/list.
- Added a Rules tab in the bottom navigation that opens Popular Rules.
- Kept Home focused on quick actions and search.
- Bat Check remains available from Home and More.

## Upload

Upload the contents of this folder to the root of your GitHub repository. The site entry file must be named `index.html`.
